About this site

This site functions as a content portal for topics related to the CIS 209 course at WCCCD. This site targets the student body of the class for the Summer 2021 semester. The course is entirely online via WCCCD Blackboard. The class does not make use of an online homework or testing engine (e.g.: Cengage MindTap, Pearson MyLab). The book for the class is available in both hard copy and an electronic edition. A free copy of the international edition can be obtained here. Do note that the international version differs in the available exercises at the end of each chapter. If you are making use of the International version, be sure to keep an eye on the assignments page of this site.

Assignments are delivered via Blackboard. I will re-post them to this site along with an image of the page from the physical book. This should answer any questions those of you using the international edition may have. Also, be sure to check out the guides section of this site. It provides useful links on setting up the development environment and additional C programming learning resources.

If you are interested in content updates or feature requests, reach out to Rob via email. I will attempt to accomodate reasonable requests when time permits.
